Creating a React App with Create React App (CRA)

Beginner 15 min

Follow these steps to create your first React app using Create React App (CRA).

1

Install Node.js & npm

node -v

Download Node.js if not installed: https://nodejs.org

2

Create the React App

npx create-react-app my-app

Replace my-app with your project name.

3

Navigate to the Project Folder

cd my-app
4

Start the Development Server

npm start

This will start the dev server and open the app in your browser (http://localhost:3000).

5

Build for Production (Optional)

npm run build

Generates a build/ folder with optimized production files.

6

Run Tests (Optional)

npm test
7

Install Additional Packages (Optional)

npm install axios react-router-dom

Creating RESTful APIs

Intermediate 20 min

Learn how to build RESTful APIs with Express.js.

1

Set up Express server

const express = require('express');
const app = express();
const PORT = process.env.PORT || 3000;

app.use(express.json());
2

Create API routes

// Get all users
app.get('/api/users', (req, res) => {
  // Logic to fetch users
  res.json(users);
});

// Get user by ID
app.get('/api/users/:id', (req, res) => {
  const user = users.find(u => u.id === req.params.id);
  res.json(user);
});
3

Start server

app.listen(PORT, () => {
  console.log(`Server running on port ${PORT}`);
});

Database Integration

Advanced 25 min

Learn how to integrate MongoDB with your application.

1

Install dependencies

npm install mongoose dotenv
2

Set up connection

const mongoose = require('mongoose');
require('dotenv').config();

mongoose.connect(process.env.MONGODB_URI, {
  useNewUrlParser: true,
  useUnifiedTopology: true
}).then(() => {
  console.log('Connected to MongoDB');
}).catch(err => {
  console.error('MongoDB connection error:', err);
});
3

Create data model

const userSchema = new mongoose.Schema({
  name: { type: String, required: true },
  email: { type: String, required: true, unique: true },
  createdAt: { type: Date, default: Date.now }
});

const User = mongoose.model('User', userSchema);
module.exports = User;

About Telusko

Empowering developers through educational excellence since 2015

Our Story

Founded in 2015, Telusko began with a simple mission: to make complex technology concepts accessible to everyone. What started as a YouTube channel has grown into a comprehensive educational platform serving millions of developers worldwide.

Our founder, Navin Reddy, a passionate educator with over 15 years of industry experience, built Telusko on the belief that quality education should be available to anyone eager to learn, regardless of their background or location.

Our Mission

At Telusko, we're committed to democratizing technical education. We strive to create learning experiences that are:

  • Accessible

    Breaking down complex concepts into digestible, easy-to-understand modules

  • Practical

    Focusing on real-world applications and industry-relevant skills

  • Community-Driven

    Building a supportive environment where learners help each other grow

What is Telusko Documentation Portal?

Telusko Documentation Portal is a comprehensive resource hub that provides detailed guides, tutorials, API references, and code examples for all Telusko products and technologies. It's designed to help developers and educators at all skill levels successfully implement and use our platform.

Is Telusko Documentation free to access?

Yes, all Telusko documentation is completely free and accessible to everyone. We believe in open access to knowledge and education. While some advanced features of our platform may require a subscription, the documentation is available to all users regardless of subscription status.

How often is the documentation updated?

Our documentation is continuously updated to reflect the latest features, best practices, and community feedback. Major updates are typically released alongside product updates, but we make smaller improvements and corrections on a daily basis. You can see the last update date at the bottom of each documentation page.

Do I need an account to access the documentation?

No, you don't need an account to access and read our documentation. However, creating a free account allows you to bookmark pages, track your progress, and contribute to the community discussions.

How do I create an account?

To create an account, click on the "Sign Up" button in the top-right corner of the page. You can register using your email address, Google account, or GitHub account. The registration process takes less than a minute, and you'll immediately gain access to all account-related features.

Can I download documentation for offline use?

Yes, registered users can download documentation in PDF format for offline reference. Look for the download icon at the top of each documentation page. We also offer a complete documentation package that can be downloaded as a single ZIP file from your account dashboard.

Where should I start if I'm new to Telusko?

We recommend starting with our "Getting Started" guide, which provides an overview of the platform and walks you through the basic setup process. After that, you can explore the "Core Concepts" section to gain a deeper understanding of how everything works together.

Are there interactive examples in the documentation?

Yes, many of our documentation pages include interactive code examples that you can edit and run directly in your browser. This allows you to experiment with different approaches and see the results immediately without having to set up a local development environment.

How can I contribute to the documentation?

We welcome contributions from the community! You can suggest edits by clicking the "Edit this page" link at the bottom of any documentation page. This will take you to our GitHub repository where you can submit a pull request. For more substantial contributions, please see our Contribution Guidelines page.

How do I get an API key?

You can generate an API key from your account dashboard. Go to "Settings" > "API Keys" and click "Generate New Key". Make sure to store your key securely as it provides access to your account and resources.

What are the API rate limits?

Rate limits vary depending on your subscription tier. Free accounts have a limit of 1,000 requests per day, while paid plans have higher limits. For detailed information, please see our API Rate Limits page in the API Reference section.

Are there SDKs available for different languages?

Yes, we provide official SDKs for JavaScript, Python, Java, Ruby, and PHP. Community-maintained SDKs are available for other languages like Go, C#, and Swift. All SDKs are documented in the "SDK Reference" section of our documentation.

What should I do if I encounter an error?

First, check the "Common Errors" section of our documentation, which covers solutions to frequently encountered issues. If you don't find a solution there, you can search our community forums or submit a support ticket through your account dashboard.

How can I debug my API requests?

We provide a request logging feature in your account dashboard that allows you to see the last 100 API requests, their status codes, and response bodies. Additionally, you can use our "API Playground" to test requests and see detailed responses in real-time.

Where can I report documentation errors or unclear sections?

If you notice an error or find a section of the documentation that's unclear, you can click the "Feedback" button at the bottom of any page. This allows you to submit detailed feedback directly to our documentation team, who will review and address your concerns.
